# 🔐 CyberSafe – Cyber Security Awareness Website

## 📌 Project Overview

CyberSafe is a simple and responsive **Cyber Security Awareness website** designed using **HTML and CSS**. The purpose of this project is to educate users about common cyber threats and provide essential safety tips to stay secure in the digital world.

This project is beginner-friendly and focuses on clean design, layout structuring, and basic web development concepts.

---

## 🎯 Features

* 🏠 **Navigation Bar** – Easy access to different sections like Home, Threats, Tips, and Contact
* 🌐 **Hero Section** – Eye-catching introduction with a call-to-action button
* ⚠️ **Cyber Threats Section** – Highlights common threats like:

  * Phishing
  * Malware
  * Password Attacks
* 💡 **Safety Tips Section** – Provides simple tips to stay safe online
* 🎨 **Modern UI Design** – Dark theme with hover effects and responsive card layout
* 📱 **User-Friendly Interface** – Clean and easy to understand

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
